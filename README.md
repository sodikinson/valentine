# Go Out - Date Proposal App

A fun and interactive web application designed for couples to propose dates in a creative way. Whether you're planning a romantic evening or a casual outing, this app makes it easy and exciting!

## Features

- **Interactive Proposal**: Ask your partner out with a personalized message.
- **Yes/No Response**: Navigate to a celebratory "Yes" page or handle rejections gracefully.
- **Simple and Responsive**: Works seamlessly on desktop and mobile devices.
- **Customizable**: Easily modify styles and content to fit your needs.

## How to Use

1. Open `index.html` in your web browser.
2. Customize the proposal message in `script.js` if desired.
3. Share the link with your partner and wait for their response!

## Technologies Used

- **HTML**: Structure of the web pages.
- **CSS**: Styling for a beautiful and responsive design.
- **JavaScript**: Interactive functionality for the proposal flow.

## Project Structure

- `index.html`: Main proposal page.
- `script.js`: JavaScript logic for interactions.
- `styles.css`: Styles for the main page.
- `yes_page.html`: Page displayed on a "Yes" response.
- `yes_style.css`: Styles for the yes page.
- `version.json`: Version information.
